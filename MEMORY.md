# Decision Log & Tech Stack

## Locked Tech Stack
- **Framework:** React 18 (via Vite for maximum client-side performance)
- **Language:** JavaScript/JSX
- **Styling:** Vanilla CSS (Strict adherence to HSL colors and 4px grid per vault rules)
- **State Management:** Zustand (for complex game loop, piece coordinates, and timers)
- **Animation/Physics:** Framer Motion (for drag-and-drop physics, confetti, and hint simulations)

## Decisions
- **Project Initiation:** Created the `african-art-puzzle` directory and scaffolded React 18.
- **Puzzle Mechanics:** Chosen a "Scattered Grid" approach over traditional jigsaw interlocking pieces to align with the "easy = less scattered" rule.
- **Art Direction:** Concept A (Earthy & Traditional: terracotta, ochre, traditional geometric patterns).
